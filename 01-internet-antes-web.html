Internet antes de la Web

Explorando fundamentos de las comunicaciones electrónicas

Telégrafo (1840)

  • Utiliza código Morse
  • Impulsos eléctricos
  • Primer medio de comunicación eléctrico
  • Mensajes enviados por cables o radiofrecuencia
  • Uso militar

ARPANET (1958)

  • Advanced Research Projects Agency Network
  • Comunicación directa entre computadoras
  • Uso militar (Departamento de Defensa)
  • 1969: Primera computadora conectada

Redes Descentralizadas

  • 1975: Telnet (Versión comercial de ARPANET)
  • Telecommunication Network: Usado para acceder a equipos remotos
  • TCP/IP (Internet Protocol Suite):
    • TCP: Transmission Control Protocol - Envío ordenado y verificado de paquetes
    • IP: Internet Protocol - Capa de red

Tim Berners-Lee

  • Científico de computadoras
  • Inventor del World Wide Web
  • Creador de HTML, URL y HTTP

World Wide Web (WWW)

  • Sistema de información para transmitir datos en Internet
  • Diseñado para ser amigable
  • 1989: Creado en el CERN
  • Incluye el protocolo HTTP

HTTP (Hypertext Transfer Protocol)

  • Transferencia de archivos de hipertexto (textos con enlaces a otros textos)
  • No guarda información de la conexión; solo transfiere archivos
  • Para almacenamiento de estado usamos cookies
